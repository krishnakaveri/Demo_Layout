Trillo Applications

Trillo Applications are customizable end-to-end applications (Open Source Code). The can be used out of the box or extended.

Trillo Text Matching

Using Vertex AI MActching Enginer,Trillo Text Matching provides end to end processing,It extracts text from files and indexes for matching.A target document for the similarities. It uses custom models for embedding.

Trilo File Manager

Trillo File manager is an application like GDrive or Dropbox on the top of Google Cloud Storage.It has an UI and supports SFTP for file transfer.

Trillo NLP

Trillo NLP extracts text from files and processes using Google Natural Language AI or custom models for analysis such as sentiment,entity recongnition,classification.

Trillo Vision AI

Using Google Vision AI or custom models,Trillo Vision AI provides end to end processing for several vision analysis such as object detection,defection,defect detection,action recongnition,etc.

Trillo Document AI

Using Google Document AI,it provides end to end processing of converting unstructure data(in documents, foems,images,etc) to structured data.

Trillo DICOM

Trillo DICOM is a service for transferring radiology images from hospitals and laboratories to Google Cloud and stone.
